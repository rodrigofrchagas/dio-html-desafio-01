Títulos - <h1> até <h6>

As tags <h1> até <h6> definem títulos nas páginas, sendo o <h1> o título principal e de maior hierarquia, o <h2> tendo uma hierarquia menor que o <h1> e assim sucessivamente, indo até o <h6>. É uma boa prática que a página tenha apenas um <h1>. Segue exemplos das tags:

Tag <p>

A tag <p> define um parágrafo, sendo comumente utilizada para blocos de texto.

Listas

Há dois tipos de listas no HTML, as listas ordenadas e as listas não ordenadas. As listas ordenadas são compostas pela tag <ol> e são visualmente representadas assim:

  1. Item 1
  2. Item 2
  3. Item 3

Já as listas não ordenadas são visualmente representadas assim:

  • Item 1
  • Item 2
  • Item 3

Todos os itens de listas devem estar envoltas da tag <li>. Além disso, é possível encapsular listas, como o exemplo a seguir:

    1. Item 1
    2. Item 2
    3. Item 3
    • Item 1
    • Item 2
    • Item 3

Links

Os links são compostas pela tag <a>, sendo um hyperlink para outra página ou uma referência para um conteúdo da mesma página. Segue a representação para o link da página inicial do Google: Google.
